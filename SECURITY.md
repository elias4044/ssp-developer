# Security Policy

## Supported versions

Only the latest version of the `master` branch is actively maintained.

## Scope

This repository contains **only the documentation website** for SchoolSoft+. It does not handle any user credentials, session tokens, or personal data.

If you believe you have found a security vulnerability in **SchoolSoft+ itself** (the main app at `ssp.elias4044.com`), please report it privately — do **not** open a public issue.

## Reporting a vulnerability

Please email **elias@elias4044.com** with:

- A description of the vulnerability
- Steps to reproduce (if applicable)
- The potential impact

You can expect an acknowledgement within **48 hours** and a resolution or status update within **7 days**.

Please do not publicly disclose the issue until it has been addressed.
